# SIMPLE_MERN_STRIPE_PAYMENT_DEMO
It's a simple example or demo project to show stripe payment
it's made by using following technologies
- ReactJS
- NodeJS
- MongoDB
- ExpressJS
- MaterialUI

## How To Run 
```
- Server
    1. Move To Server Directory
    cd server/

    2. Install Packages
    yarn

    3. start Server App
    nodemon src/server.js 

- Client    

    1. Move To Client Directory
    cd Client/

    2. Install Packages
    yarn

    3. start Server App
    yarn start
```
